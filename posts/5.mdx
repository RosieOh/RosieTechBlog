# 광진구 빅데이터 분석 공모전 회고

서울시 광진구에서 주최한 빅데이터 분석 공모전에 ‘팀 정이가요’로 참가하여, 공공 CCTV 데이터와 주정차 단속 데이터를 활용한 **차량 불법 주정차 패턴 분석 및 안전도 가시화 프로젝트**를 수행하였습니다.

## 📊 프로젝트 개요

- **목표**: 주정차 단속 데이터, CCTV 위치, OCR을 기반으로 차량의 불법 주정차 가능성을 예측하고, 위험도에 따라 가시화하는 시스템 개발
- **활용 데이터**
  - 서울시 공공 CCTV 위치 정보
  - 광진구 주정차 단속 이력
  - OCR로 추출한 차량 번호판 및 위치/시간 데이터
- **주요 분석 방법**
  - Spearman 상관 분석
  - 단일/다중 회귀 분석
  - 차량 정지 판단 알고리즘 설계 및 위험도 분류

## 🛠️ 사용 기술 스택

- Python (pandas, sklearn, matplotlib, seaborn)
- OCR: EasyOCR
- Jupyter Notebook
- 데이터 시각화 도구: PowerPoint, Tableau 대체

## 😖 어려웠던 점

- OCR 결과가 완벽하지 않아 차량 번호판 인식률이 낮았고, 이로 인해 정확한 차량 추적에 어려움이 있었습니다.
- 정지 차량 판단 알고리즘 구현 시 시간 간격 설정과 속도 계산 기준을 반복적으로 보완해야 했습니다.
- 회귀 분석 모델 해석과 다중공선성 문제를 해결하는 과정에서 통계적 지식이 부족해 많은 학습이 필요했습니다.

## 📚 배운 점

- 실제 행정 데이터를 다뤄보며 공공 데이터의 전처리 중요성을 절실히 깨달았습니다.
- 회귀 분석의 해석 및 적용 방법, 특히 다중 회귀 분석의 한계와 적합도를 이해하게 되었습니다.
- OCR과 객체 추적 기술을 통해 정적 데이터를 동적 분석 데이터로 전환하는 과정을 경험할 수 있었습니다.

## 💡 느낀 점

이번 공모전은 단순한 분석을 넘어 **도시 안전이라는 실질적 문제 해결에 기여**한다는 점에서 매우 의미 있는 경험이었습니다. 한정된 시간과 자원 속에서도 팀원들과 협업하며 문제를 정의하고 솔루션을 설계하는 과정 자체가 값진 배움이었습니다.

> “데이터는 숫자에 머물지 않고, 실생활의 위험을 줄이는 도구가 될 수 있다”는 사실을 실감한 프로젝트였습니다.
